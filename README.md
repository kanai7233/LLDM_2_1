A continuació fem un repàs un per un de tots els conceptes que inclou la pàgina\
web que hem vist anteriorment:

HTML:\
- Encapçalament mitjançant l'etiqueta <header> emprant l'element de grup de\
capçaleres <hgroup>:\
o Agrupació d'encapçalaments.\
- Llistes no ordenades amb els elements <ul> y <li>\
- Element ancla amb l'atribut <a>:\
o Crea un enllaç a altres pàgines de internet.\
- Element <href>:\
o Defineix el destí on apunta l'enllaç.\
- Element <section>\
o Determinar quin contingut correspon a cada part de un esquema.
- Element (heading) <h1> ... <h6>:\
o Etiquetes de capçalera, permet especificar un tamany de lletra per el\
text que conté i provoca un salt de línea.

- Element <time>:\
o Representa un període específic en el temps, pot incloure l'atribut\
"datetime" per convertir les dates en un format intern legible per un\
ordenador.\
- Element <img>:\
o Utilitzat per incrustar imatges, va acompanyat de dos dels atributs\
principals de les imatges:\
- <src>: Defineix la ruta o URL de la imatge.\
- <alt> : Proporciona text alternatiu que es mostra si la imatge no\
es pot carregar.

- Element <footer>:\
o Representa el peu de pàgina de un document o de una secció.\
- Formulari de registre <form> y </form> i dins aquest formulari tenim:\
o Etiqueta <input> per crear camps de entrada dins un formulari.\
Recordeu que el tipus de dada que s'ingressa depen del valor del\
atribut "type".

CSS:\
- Relacionar fulla de estil CSS amb document HTML:\
o Dedins de la capçalera amb la etiqueta <style> </style>, d'aquesta\
forma es converteix amb un element de capçalera i s'aplica a tot\
l'arxiu.\
o Amb un arxiu extern (Recomanat per tenir-lo separat i anar fent\
proves), amb aquest mètode s'ha de guardar la fulla de estils externa\
amb la extensió .css i després s'utilitza una etiqueta de enllaç per\
inclourer-la dins l'arxiu HTML.

- Selectors:\
o Selector universal (*):\
- Selecciona tots els elements de un document.\
o Selector id:\
- Selecciona elements amb base al seu atribut ID, important\
recordar que sols funciona quant el valor donat al selector es\
idèntic a l'atribut al que es vol referir:\
- #nom de ID\
- Es a dir dins HTML definim un id="nav" i a la fulla de\
estil de CSS li donam el format amb #nav.\
